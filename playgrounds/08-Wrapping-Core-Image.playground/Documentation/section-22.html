Here we use Swift's generics to make the definition more flexible than we might need in this specific example. The composeFunctions function takes two parameters: one function of type B -> C and another function of type A -> B (with A, B, and C may be any type). From those two functions we construct a new function of type A -> C, that passes its argument arg of type A to the second function, producing a value of type B.
